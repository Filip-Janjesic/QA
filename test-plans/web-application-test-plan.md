# Test Plan – Web Application

## Objective

The objective of this test plan is to verify that the web application functions correctly and meets the specified requirements.

---

## Scope

The following features will be tested:

- User authentication
- User profile management
- API endpoints
- Input validation

---

## Test Types

The following testing types will be performed:

- Functional testing
- API testing
- Negative testing
- Basic security validation

---

## Test Environment

- Browser: Chrome, Firefox
- Operating System: Windows
- Environment: QA / Staging

---

## Test Data

Test data will include:

- Valid user credentials
- Invalid login attempts
- Non-existing user IDs
- Incorrect input formats

---

## Entry Criteria

- Application is deployed to QA environment
- Test accounts are available

---

## Exit Criteria

Testing is completed when:

- All critical test cases are executed
- All blocking bugs are resolved
